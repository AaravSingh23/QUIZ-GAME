# 🧠QUIZ GAME

## 📖 Description

The **Python Computer Quiz Game** is a command-line application developed using **Python**.

It is designed to test users' knowledge of **Computer Science**, **Python Programming**, **Operating Systems**, **Networking**, **Algorithms**, and **Artificial Intelligence**.

The game provides an interactive quiz experience through a simple and user-friendly terminal interface.

Users can choose whether they want to start the quiz before answering the questions.

The quiz contains **20 carefully selected multiple-topic questions** covering fundamental and intermediate-level computer concepts.

The application instantly checks each answer and provides immediate feedback.

Correct answers increase the user's score, while incorrect responses are identified instantly.

At the end of the quiz, the program displays:

* ✅ Total Correct Answers
* 📊 Final Score
* 📈 Percentage Obtained

This project demonstrates the practical implementation of core Python programming concepts, including:

* 🐍 Variables & Data Types
* ⌨️ User Input Handling
* 🔀 Conditional Statements (`if`, `else`)
* 🔁 Sequential Program Flow
* ➕ Score Calculation
* 📊 Percentage Calculation
* 🖥️ Command Line Interface (CLI)

The project helps improve logical thinking, user interaction, and basic programming skills through an engaging quiz application.
